Descrição do Projeto

BookPrizes™ é uma plataforma inovadora dedicada à promoção da leitura, oferecendo uma experiência dinâmica, divertida e recompensadora para os usuários. Através de desafios literários, análises críticas e prêmios exclusivos, a plataforma visa incentivar e premiar a jornada de leitura dos participantes.

Estrutura do Projeto

1. index.html

Arquivo HTML principal que estrutura o conteúdo da página, incluindo a navegação, seções de conteúdo e funcionalidades como carrossel de imagens, formulário e depoimentos.

2. assets/

Diretório contendo arquivos estáticos como imagens e o ícone do site.

3. css/

Diretório contendo arquivos de estilo CSS:

  • style.css: Estilos principais da plataforma.
  • animacao.css: Estilos para animações na página.
  • responsivo.css: Estilos para tornar a plataforma responsiva.
  • carrosel.css: Estilos específicos para o carrossel de imagens.
  • hover.css: Estilos para efeitos de hover nos elementos interativos.

4. fontawesome-free-6.4.2-web/

Diretório com os arquivos de ícones do Font Awesome, usados para ícones sociais e outros elementos visuais.

5. js/

Diretório com arquivos JavaScript:

  • main.js: Funcionalidades gerais do site.
  • gridScript.js: Script para gerenciamento da galeria de imagens.
  • responsivo.js: Script para controle da responsividade do site.
  • animacao.js: Script para animações de elementos.

6. Formulários e Interações

O formulário de envio de dados inclui campos para nome, email e telefone. Ele utiliza validação básica para garantir que os dados sejam inseridos corretamente antes de serem enviados.

Funcionalidades

    Carrossel de Imagens

  1. Um carrossel interativo que exibe produtos da plataforma, permitindo que o usuário navegue entre as imagens.

    2. Galeria de Imagens

  2. Uma grade de imagens clicáveis que abre uma visualização em tela cheia em um modal.

    3. Depoimentos

  3. Carrossel de depoimentos de usuários, permitindo a navegação entre as opiniões de participantes sobre a plataforma.

    4. Desafio da Semana

  4. Seção promocional destacando o desafio literário atual, incentivando os usuários a participar.

    5. Rodapé com Contatos

  5. Inclui informações de contato e links para redes sociais.

Responsividade

O projeto foi desenvolvido com foco em responsividade, utilizando arquivos CSS e JavaScript específicos para garantir que o layout se adapte corretamente a diferentes dispositivos e tamanhos de tela.

Instruções de Uso

Para visualizar e testar o projeto localmente:

  1. Clone ou baixe o repositório do projeto.
  2. Abra o arquivo index.html em um navegador da sua preferência.

Requisitos

Este projeto utiliza o Bootstrap para estilização e o Font Awesome para ícones. Certifique-se de que o navegador tenha acesso à internet para carregar os recursos externos.

Tecnologias Utilizadas

  • HTML5: Estruturação do conteúdo da página.
  • CSS3: Estilização da página, incluindo responsividade e animações.
  • JavaScript: Funcionalidade interativa, como controle do carrossel e visualização de imagens em modal.
  • Bootstrap 5: Framework CSS para responsividade e componentes prontos.
  • Font Awesome: Biblioteca de ícones para melhorar a interface do usuário.
